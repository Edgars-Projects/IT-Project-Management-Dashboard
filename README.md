# IT Project Management Dashboard

This project provides an Excel-based project management dashboard designed to support IT project coordination and monitoring.

The dashboard demonstrates how project coordinators track tasks, risks, issues, and milestones throughout the lifecycle of a project.

---

## Project Overview

The dashboard simulates the management of a cloud migration implementation project. It provides a structured approach to tracking project progress and identifying potential risks that may impact delivery timelines.

The tool is designed to reflect common practices used in IT project environments.

---

## Key Features

- Task tracking and status monitoring
- Risk identification and mitigation planning
- Issue logging and resolution tracking
- Milestone monitoring
- Centralised project overview

---

## Dashboard Components

The Excel workbook contains the following sheets:

### Project Overview
Provides a summary of the project including project manager, timeline, and current project status.

### Task Tracker
Tracks project activities, assigned teams, deadlines, and task progress.

### Risk Register
Identifies potential project risks and defines mitigation strategies.

### Issue Log
Tracks problems that occur during project execution and their resolution status.

### Milestones
Monitors major project checkpoints and delivery targets.

---

## Purpose of the Project

This project demonstrates practical IT project coordination skills including:

- project planning
- risk management

- project monitoring

These are core activities typically performed by IT project coordinators and project managers.
